Política de Privacidade

Extensão: CRTI Launcher – Grupo JR
Desenvolvedor: Grupo JR
Última atualização: 24 de abril de 2026

Esta Política de Privacidade descreve como a extensão CRTI Launcher – Grupo JR coleta, usa e protege as informações dos usuários.

1. Sobre a Extensão

O CRTI Launcher é uma extensão para o navegador Google Chrome desenvolvida pelo Grupo JR. Sua finalidade exclusiva é automatizar o lançamento de dados de Notas Fiscais (NF) no sistema ERP interno da empresa, denominado CRTI.

2. Dados Coletados

A extensão acessa e processa as seguintes informações, exclusivamente durante a operação de lançamento de NF:

  • Dados de Notas Fiscais: número, série, data de emissão, valores, fornecedor/destinatário e demais campos necessários ao lançamento no ERP CRTI.
  • Dados de navegação na aba ativa: a extensão lê o conteúdo da aba do navegador onde o usuário está realizando o lançamento, exclusivamente para identificar e preencher os campos do formulário do ERP.

A extensão não coleta dados pessoais sensíveis, não acessa histórico de navegação, senhas, dados bancários ou qualquer informação além das descritas acima.

3. Uso dos Dados

Os dados acessados pela extensão são utilizados unicamente para:

  • Preencher automaticamente os campos do ERP CRTI com as informações da Nota Fiscal;
  • Confirmar e finalizar o lançamento da NF no sistema interno.

Nenhum dado é enviado a servidores externos, terceiros ou quaisquer outros destinos fora do ambiente do ERP CRTI da empresa.

4. Armazenamento de Dados

A extensão não armazena nenhum dado do usuário de forma persistente. Todos os dados processados são utilizados em memória apenas durante a execução da tarefa e descartados imediatamente após a conclusão do lançamento.

5. Compartilhamento de Dados

Os dados processados pela extensão não são compartilhados com nenhum terceiro, empresa parceira, serviço de análise ou publicidade. O tráfego de dados ocorre exclusivamente entre o navegador do usuário e o sistema ERP CRTI interno da empresa.

6. Permissões Utilizadas

A extensão solicita apenas as permissões estritamente necessárias para seu funcionamento:

  • activeTab: para leitura do conteúdo da aba onde o ERP está aberto;
  • scripting: para interagir com os campos do formulário do ERP;
  • storage (se aplicável): para armazenar configurações locais da extensão, sem dados pessoais.

7. Segurança

O Grupo JR adota medidas técnicas adequadas para garantir a segurança das informações processadas pela extensão, incluindo restrição de acesso ao código-fonte e uso restrito de permissões do navegador.

8. Direitos do Usuário

Em conformidade com a Lei Geral de Proteção de Dados (LGPD – Lei nº 13.709/2018), os usuários têm direito a:

  • Solicitar informações sobre os dados tratados;
  • Solicitar a exclusão de quaisquer dados eventualmente armazenados;
  • Revogar o consentimento a qualquer momento, desinstalando a extensão.

9. Público-Alvo

Esta extensão é destinada exclusivamente a colaboradores e usuários internos do Grupo JR que utilizam o ERP CRTI. Não é destinada ao público geral nem a menores de idade.

10. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente. Notificaremos os usuários sobre mudanças significativas por meio dos canais internos do Grupo JR. O uso continuado da extensão após alterações implica concordância com a política atualizada.
